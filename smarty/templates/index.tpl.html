Cartiki is a location database that anyone can edit. The name is a combination of cartography and wiki.

Anything that has a generally accepted name and is relatively stationary belongs in Cartiki. So, things like countries, cities, campuses, buildings, and rooms belong in Cartiki. Vehicles, furniture, and people do not.

It stores the following about each unique location:

  • names for the location
  • geographical bounds for the location
  • its place in the hiearchy of locations

To add a location, find its parent location, make sure you are in edit mode, and click "add child".
